# Web Display of Segment And NER
一些分词和实体命名识别的方法的网页展示

## Usage:
  the website is coded based on python-django
 ```
  - pip install django
 ```
  the way to start server
  ```
  - python manage.py runserver 0.0.0.0:8080
  ```
